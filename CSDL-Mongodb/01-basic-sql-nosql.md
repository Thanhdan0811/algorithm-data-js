# SQL 
- Học 1 ngôn ngữ dùng đc cho nhiều csdl khác nhau
- schema chặt chẽ.
- khuyển khích tiêu chuẩn hóa.

# NoSQL
- Lưu dưới dạng json với các cặp key-value.
- Không cần schema.
- Hiệu năng tốt, mở rộng dễ.

- collection như là 1 bảng.
- document như là 1 row trong SQL.
