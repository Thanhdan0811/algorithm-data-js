# Array Big O
Searching => O(N)
Access => O(1)
Insertion => It depends : chèn ở đầu thì là O(N), cuối thì là O(1)
Removal => It depends  

# Array methods.
push => O(1)
pop => O(1)
shift => O(N)
unshift => O(N)
concat => O(N)
slice => O(N)
splice => O(N)
sort => O(n*logN)
forEach/map/filter/reduce/etc => O(N)
